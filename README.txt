Minimal Telegram bot with one button.

Files:
- simple_bot.py    : the bot script (contains a placeholder token)
- requirements.txt : pip requirements

Usage:
1. Replace BOT_TOKEN in simple_bot.py with your bot token (or leave placeholder if for testing).
2. Install dependencies:
     pip install -r requirements.txt
3. Run:
     python simple_bot.py
4. In Telegram, send /start to your bot. Tap the button.

Note: Keeping tokens in code is insecure; this example uses a placeholder token.
